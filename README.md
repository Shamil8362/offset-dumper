# Offset Dumper

The Offset Dumper is a tool designed to extract offsets from binary files for analysis.

## Features
- Lightweight and easy to use
- Supports multiple binary formats
- Configurable output options

## Installation
To install the Offset Dumper, clone the repository and compile the source code:

```bash
git clone https://github.com/Shamil8362/offset-dumper.git
cd offset-dumper
make
```

## Usage
Run the tool using the following command:

```bash
./offset-dumper [options] <file>
```

## Options
- `-h`, `--help` : Show help information
- `-o`, `--output` : Specify output format (e.g., json, xml)
